# Truth Vault

Independent investigative journalism platform built with React, Vite, and Firebase.

## Quick Start

1. Copy `.env.example` to `.env` and fill in your Firebase credentials
2. Run `npm install`
3. Run `npm run dev`

## Deployment

Push to GitHub main branch to trigger GitHub Pages deployment via GitHub Actions.

## Firebase Setup

1. Create project in Firebase Console (Spark/Free plan)
2. Enable Authentication (Email/Password + Google)
3. Create Firestore Database
4. Enable Storage
5. Deploy rules and indexes

See the full setup guide in the project documentation.
